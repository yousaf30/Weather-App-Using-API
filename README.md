# 🌤️ Weather App

A modern, responsive Weather Application built using **HTML, CSS, and JavaScript**.  
This project uses the **OpenWeatherMap API** to fetch real-time weather data.

---

## 🚀 Features

- 🔍 Search weather by city & country code
- 📍 Get weather using current location (Geolocation API)
- 🌡️ Real-time temperature display
- 💧 Humidity, wind speed, and weather condition
- 🌤️ Dynamic weather icons (Clear, Rain, Clouds, Snow, etc.)
- 🎨 Weather-based dynamic background images
- 🎯 Color-coded weather values for better UI experience
- 📱 Fully responsive design (mobile + desktop)

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeatherMap API
